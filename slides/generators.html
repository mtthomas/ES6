Generators


function* createId(){
  var index = 0;
  while(true){
    yield index++;
  }
}

var generator = createId();
console.log(generator.next().value) //0
console.log(generator.next().value) //1
console.log(generator.next().value) //2
  • Allow you to easily use iterators (outside the scope of this talk)
  • Generators are created using function *
  • yield pauses and resumes a generator, releases the single JS thread
